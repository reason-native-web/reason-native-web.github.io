Core concepts

Morph_client is built on top of Opium_core that provides us with Filters and Services. The Morph_client is just a Opium_core.Serivce.t so it takes a Morph_core.Request.t and returns a Morph_core.Response.t wrapped in a Lwt.t.

To read more about Filters and Services I recommend going to the Opium_core documentation.

For examples go to the reason-native-web homepage or look in the examples folder of this project.
